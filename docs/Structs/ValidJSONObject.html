  • associate(key:value:)

    Sets (i.e. associates) value with a given key. Blows up if value is not valid JSON.

    Throws

    JSONError.invalid if value isn’t valid JSON.

    Note

    We’d like to model this as a subscript instead, but there’s currently no way to make a subscript setter throw.

  • associateIfValid(key:value:)

    Sets (i.e. associates) value for a given key, but only if value is valid JSON.

    Note

    It’d make sense to model this as a subscript, but subscripts are generally thought of as assignment in Swift. A conditional subscript would be weird and unexpected behavior.
