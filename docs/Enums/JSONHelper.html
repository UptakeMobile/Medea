JSONHelper

public enum JSONHelper

JSONHelper is a bag of functions for converting to and from JSON representations. It’s a very thin wrapper around JSONSerialization.

  • anyJSON(fromFileNamed:extension:bundle:)

    Decodes contents of UTF-8 as JSON.

    Note

    While this will parse files a file as a string, a numbers or even just null, there are probably more efficient ways of doing this than going through a JSON deserializer.

    Seealso
